/* コンポーネントとprops */

コンポーネントとは？
・見た目と機能を持つUI部品
・コンポーネントを組み合わせてページを作る
・大きく２種類のコンポーネントに分かれる
    Class Component (クラスコンポーネント) -> このクラスコンポーネントは現在ほとんど使われなくなってきてる。
    Functional Component (関数コンポーネント) -> Javascriptの関数ライクの書き方でわかりやすい。
・１コンポーネントにつき １ファイル。
    別ファイルに分けることで、コードが読みやすくなる？

結論
Reactを使うときは、コンポーネントごとにファイルを分けよう。


コンポーネントの基本的な使い方
・ファイル名は大文字
・子コンポーネントでexport
・親コンポーネントでimport
・別ファイルでexportされたコンポーネントを別のファイルでimportすることで
    別々に分けたコンポーネントを使うことができる。


propsとは？
親コンポーネントから、子コンポーネントにデータを渡すときに使う。
引数みたいな役割を持つと思う。
propsで渡せるデータ。
・propsのデータは、{}でかこう
・文字列、数値、真偽値、配列、オブジェクト、日付なんでもok
・変数を渡すことも可能
